# Documentação do Repositório

## Configuração Inicial
Para configurar o Databricks com este repositório:
1. Crie um workspace no Databricks.
2. Conecte ao GitHub: Vá em User Settings > Git Integration > Add Git Provider.
3. Clone este repositório.
